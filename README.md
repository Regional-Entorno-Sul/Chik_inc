# Chik_inc
Procura por inconsistência no resultado de exames de Febre Chikungunya usando a base de dados importada do SINAN Online.

## Como funciona?  

A ferramenta procura pelos resultados da sorologia IgM e do exame PRNT, ambos relativos ao agravo Chikungunya.

![x](/ficha_pic1.jpg)  

A inconsistência aparece quando os resultados são registrados no sistema como não reagentes, inconclusivo ou não realizados e o caso é classificado como Febre Chikungunya usando o critério confirmação/descarte.
